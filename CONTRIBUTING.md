# Contributing

Contributions are welcome through issues, discussions, and pull requests.

This repository is an idea-first design pattern / schema proposal for Action Branch Cache (ABC). It is not a production framework and does not claim to provide real-time behavior, measured performance improvements, or safety guarantees. Please keep contributions aligned with that boundary.

## Welcome contributions

- Clearer examples
- Schema suggestions
- Terminology improvements
- References to related ideas
- Small illustrative demos

## Out of scope

- Claiming real-time safety
- Turning ABC into a production agent framework without clear boundaries
- Adding examples outside the stated embodied, browser, and office automation scope

When proposing changes, prefer small, reviewable updates that make the concept easier to discuss.
