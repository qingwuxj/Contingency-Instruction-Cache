# Contributing

Contributions are welcome through issues, discussions, and pull requests.

This repository is a design note / schema proposal for Contingency Instruction Cache (CIC). It is not a production framework and does not claim to provide real-time behavior or safety guarantees. Please keep contributions aligned with that boundary.

## Welcome contributions

- Clearer examples
- Schema suggestions
- Terminology improvements
- References to related ideas
- Small illustrative demos

## Out of scope

- Claiming real-time safety
- Turning CIC into a production agent framework without clear boundaries
- Adding examples outside the stated embodied, browser, and office automation scope

When proposing changes, prefer small, reviewable updates that make the concept easier to discuss.
